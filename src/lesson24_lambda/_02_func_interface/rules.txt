1. Лямбда выражения может содержать 0 или более входных параметров
    () -> 27;
    (String s) -> {sout(s);}
    (int v1, int v2) -> {return v1 + v2;}
2. Тип параметров может быть явно объявлен или выведен из значения параметров
    (String str) ->{sout(str);}
        ==
        s -> {sout(s);}
3. Если  параметров нету или их больше одного, то скобки обязательны
    (v1, v2) -> return v1 + v2;
    (int v1, int v2) -> return v1 + v2;
    () -> 27;
4. Нет необходимости объявлять один параметр в скобках... но нельзя явно указывать тип параметра
    v1 -> return 2 * v1;
    (int v1) -> return 2 * v1;
5. Нет необходимости  использовать {} и return если тело состоит из одного выражения
    () -> 45;
    (int v1, int v2) -> v1 + v2;
6. Если тело состоит из больше 1 выражения, то {} и return - обязательны
    () -> {
        sout("Qwerty = 4");
        return 4;
    }
7. Если ключевое слово return отсутствует, то возвращаемый тип может быть void
    () -> sout("Hello world");
    () -> {
          sout("Qwerty = 4");
          return;
    }
